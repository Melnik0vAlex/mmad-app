# MMAD Calculator (Andersen Cascade Impactor)

Графическое приложение для расчёта **MMAD (Mass Median Aerodynamic Diameter)** и анализа
**массового распределения аэрозольных частиц по аэродинамическому диаметру**
по данным каскадного импактора Андерсона (Andersen Cascade Impactor).

Приложение ориентировано на учебные, научные и прикладные задачи
в области ингаляционной токсикологии.

---

## ✨ Основные возможности

- 📊 Расчёт **MMAD (d50)** и ключевых квантилей: d10, d16, d84, d90
- 📐 Расчёт **GSD**, **Span**, **FPF (< 5 µm)**
- 📈 Построение графиков:
  - накопительная кривая (*cumulative undersize*)
  - лог-пробит график
  - столбчатое распределение массы по ступеням
  - дифференциальная массовая плотность по ln(d)
- 🧮 Логнормальная аппроксимация распределения
- 💾 Сохранение расчётов и исходных данных в **SQLite**
- 📂 Загрузка сохранённых расчётов из базы данных
- 📤 Экспорт результатов и графиков
- 🪟 Современный GUI на **PySide6 (Qt Widgets)**

---

## 🧪 Используемые методы

В программе реализованы **два независимых метода** расчёта масс-медианного аэродинамического диаметра (MMAD) по данным каскадного импактора Андерсона.
Оба подхода основаны на анализе накопленного массового распределения аэрозольных частиц (APSD — Aerodynamic Particle Size Distribution).

### 1️⃣ Метод накопительного массового распределения (Cumulative Undersize)

Основной метод основан на анализе накопленной доли массы частиц:

F(d) = Σ m(d_i < d) / Σ m · 100 %

где:

- F(d) — доля массы частиц с диаметром меньше d (%),
- m — масса, осевшая на ступенях импактора.

По накопительной кривой определяются квантильные диаметры:

- D10 — 10-й процентиль,
- D16 — инженерный квантиль,
- **D50 (MMAD)** — масс-медианный аэродинамический диаметр,
- D84,
- D90.

Геометрическое стандартное отклонение:

GSD = sqrt(D84.13 / D15.87)

Ширина распределения:

Span = (D90 − D10) / D50

Также рассчитывается доля мелкодисперсной фракции:

FPF(< d_cut) = F(d_cut)

---

### 2️⃣ Метод наименьших квадратов (Lognormal Least Squares)

Предполагается, что массовое распределение аэрозоля является логнормальным.

Для каждой ступени используется репрезентативный диаметр:

d_i = sqrt(d_low · d_high)

Вводится преобразование вероятности:

p = cumulative mass fraction

z = Φ⁻¹(p)

x = arcerf(2p − 1) = z / √2

После чего выполняется линейная регрессия:

ln(d) = m · x + c

где:

- m — наклон регрессии,
- c — свободный член.

Параметры распределения:

D50 = exp(c)

σ(lnD) = m / √2

K = 1 / m

Качество аппроксимации оценивается статистиками линейной регрессии:

- R² — коэффициент детерминации,
- SE_yx — стандартная ошибка регрессии,
- F — F-статистика значимости модели.

---

### 3️⃣ Log–Probit представление распределения

Кумулятивная кривая также анализируется в координатах log–probit:

probit = Φ⁻¹(p) + 5

probit = a · log10(d) + b

Данный подход позволяет:

- визуально проверить логнормальность распределения,
- выявлять выбросы экспериментальных точек,
- контролировать корректность измерений APSD.

---

### 4️⃣ Контроль согласованности методов

В программе одновременно рассчитываются:

- MMAD (cumulative undersize method)
- MMAD (least squares lognormal fit)

Согласие результатов двух методов служит критерием корректности экспериментальных данных и применимости логнормальной модели аэрозоля.

---

### 5️⃣ Оценка качества аппроксимации

Используются статистические показатели:

- RMSE — среднеквадратичная ошибка (в probit-координатах),
- R² — доля объяснённой дисперсии,
- F-статистика — проверка значимости регрессии.

Высокие значения R² и F подтверждают логнормальный характер распределения частиц.

- Вкладка **Новый расчёт**:

## 🚀 Установка и запуск (из исходников)

```bash
git clone https://github.com/Melnik0vAlex/mmad-app.git
cd mmad-app

python -m venv .venv
source .venv/bin/activate   # Windows: .venv\Scripts\activate

pip install -r requirements.txt
pip install -e .

python -m mmad_app
```
