# MMAD Calculator (Andersen Cascade Impactor)

Графическое приложение для расчёта **MMAD (Mass Median Aerodynamic Diameter)** и анализа
**массового распределения аэрозольных частиц по аэродинамическому диаметру**
по данным каскадного импактора Андерсона (Andersen Cascade Impactor).

Приложение ориентировано на учебные, научные и прикладные задачи
в области ингаляционной токсикологии.

---

## ✨ Основные возможности

- 📊 Расчёт **MMAD (d50)** и ключевых квантилей: d10, d16, d84, d90
- 📐 Расчёт **GSD**, **Span**, **FPF (< 5 µm)**
- 📈 Построение графиков:
  - накопительная кривая (*cumulative undersize*)
  - лог-пробит график
  - столбчатое распределение массы по ступеням
  - дифференциальная массовая плотность по ln(d)
- 🧮 Логнормальная аппроксимация распределения
- 💾 Сохранение расчётов и исходных данных в **SQLite**
- 📂 Загрузка сохранённых расчётов из базы данных
- 📤 Экспорт результатов и графиков
- 🪟 Современный GUI на **PySide6 (Qt Widgets)**

---

## 🧪 Используемые методы

- Анализ кумулятивного массового распределения частиц по аэродинамическому диаметру (cumulative undersize mass distribution)
- Лог–пробит аппроксимация кумулятивного массового распределения аэрозольных частиц
- Логнормальная модель массового распределения аэрозольных частиц по аэродинамическому диаметру
- Оценка качества аппроксимации распределения по среднеквадратичной ошибке (RMSE)

---

## 🖥 Интерфейс

- Вкладка **Новый расчёт**:
  - ввод данных по ступеням импактора
  - расчёт и визуализация
- Вкладка **База данных**:
  - история расчётов
  - загрузка ранее сохранённых экспериментов
- Панель **Результаты расчетов**:
  - численные параметры APSD
  - MMAD, GSD, FPF и др.
  - графики функции и распределений

## 📦 Стек технологий

- **Python 3.11+**
- **PySide6 (Qt Widgets)**
- **matplotlib**
- **NumPy**
- **SQLite**

> ⚠️ SciPy намеренно не используется — все расчёты реализованы
> на NumPy и стандартной библиотеке Python,
> что существенно уменьшает размер сборки.

## 🚀 Установка и запуск (из исходников)

```bash
git clone https://github.com/Melnik0vAlex/mmad-app.git
cd mmad-app

python -m venv .venv
source .venv/bin/activate   # Windows: .venv\Scripts\activate

pip install -r requirements.txt
pip install -e .

python -m mmad_app
```

---

## 🪟 Сборка в exe (Windows)

Пример компактной сборки с PyInstaller:

```powershell
pyinstaller --noconfirm --clean --windowed --name mmad-app --paths src ^
  --collect-submodules PySide6.QtCore ^
  --collect-submodules PySide6.QtGui ^
  --collect-submodules PySide6.QtWidgets ^
  --collect-data matplotlib ^
  --collect-submodules matplotlib.backends ^
  src\mmad_app\__main__.py
```

## 💾 Хранение данных

- Результаты и исходные данные сохраняются в SQLite
- База данных располагается рядом с exe-файлом
- Каждому расчёту соответствует уникальный шифр пробы
